## Agent skills

### Issue tracker

Issues and PRDs for this repo live in GitHub Issues for `ymoruhyi/dotfiles`. See `docs/agents/issue-tracker.md`.

### Triage labels

This repo uses the default triage label vocabulary: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, and `wontfix`, plus `bug` and `enhancement` categories. See `docs/agents/triage-labels.md`.

### Domain docs

This repo uses a single-context domain-doc layout. Read root `CONTEXT.md` and relevant `docs/adr/` files if they exist. See `docs/agents/domain.md`.
